conditional statements

In Python, **conditional statements** allow you to execute certain blocks of code based on whether a condition is true or false. These statements help control the flow of your program. The most commonly used conditional statements are `if`, `elif`, and `else`.

if statement:

The `if` statement is used to test a condition. If the condition is true, the code block inside the `if` statement will be executed.

  • Syntax:

    • if condition:
    # Code block to execute if the condition is True

    x = 10
    if x > 5:
    print("x is greater than 5")

    Output:
    x is greater than 5

    else statement:

    The `else` statement follows the `if` statement and runs if the condition in the `if` is false.

  • Syntax:

    • if condition:
    # Code block to execute if the condition is True
    else:
    # Code block to execute if the condition is False

    x = 3
    if x > 5:
    print("x is greater than 5")
    else:
    print("x is not greater than 5")

    Output:
    x is not greater than 5

    elif statement:

    The `elif` (short for "else if") statement allows you to check multiple conditions. It follows an `if` statement and executes only if the `if` condition is false, but the `elif` condition is true. You can have multiple `elif` statements in a chain.

  • Syntax:

    • if condition1:
    # Code block to execute if condition1 is True
    elif condition2:
    # Code block to execute if condition2 is True
    else:
    # Code block to execute if all previous conditions are False

    x = 10
    if x < 5:
    print("x is less than 5")
    elif x == 10:
    print("x is equal to 10")
    else:
    print("x is greater than 5 but not 10")

    Output: x is equal to 10

    Nested if statements:

    You can also have `if` statements inside other `if` statements. This is known as a **nested if**.

    x = 15
    if x > 10:
    if x < 20:
    print("x is between 10 and 20")
    else:
    print("x is greater than or equal to 20")
    else:
    print("x is 10 or less")

    Output:
    x is between 10 and 20

    Conditional Expressions (Ternary Operator):

    In Python, you can use a conditional expression to evaluate a condition in a single line. This is also called a **ternary operator**.

  • Syntax:

    • value_if_true if condition else value_if_false

    x = 7
    result = "Positive" if x > 0 else "Negative"
    print(result)

    Output:
    Positive

    Summary of Conditional Statements:

  • `if`: Tests a condition and executes a block of code if the condition is true.
  • `elif`: Checks another condition if the previous `if` (or `elif`) condition is false.
  • `else`: Executes a block of code if none of the conditions are true.
  • You can nest `if` statements inside one another.
  • Python also has a compact conditional expression using the ternary operator.
